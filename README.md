# BCSSJ Modern Site Concept

This directory contains a modern static website concept for BCSSJ inspired by community-focused Bengali organization sites such as Gharoaa and Pragati Philly.

## Files
- `index.html` — homepage
- `about.html` — organization overview
- `events.html` — annual events page
- `membership.html` — membership page
- `festival.html` — Durga Pujo landing page
- `contact.html` — contact / engagement page
- `css/styles.css` — shared styling

## Preview locally
From this directory, run:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000` in a browser.

## Notes
This is designed as a visual/content prototype that can either:
1. be expanded into a standalone site, or
2. be translated page-by-page into Squarespace using the SOP.
